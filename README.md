## 📌 **About**

Advanced WhatsApp bot with message protection, status automation, and smart triggers. Built with Baileys. Special for music.

---

## ⚡ **Quick Install**

```bash
git clone https://github.com/Lavvordev/HavanaAi.git
cd HavanaAi
npm install
npm start
```
